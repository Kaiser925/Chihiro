## 介绍

Chihiro(千寻)是一款的简单搜索引擎。 目的是打造一个**IT First, Privacy First**的搜索网站。

当前已完成前端搭建，以及基本的爬虫开发。如果对此感兴趣，欢迎PR，fork或者star。
